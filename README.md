## fastposter电商海报生成器

> 目标：一分钟完成海报开发

🔥🔥🔥 fastposter海报生成器是一款快速开发海报的工具，只需要上传一张海报底图，然后在对应的位置放上组件（`文字`、`图片`、`二维码`、`头像`），即可生成调用的代码。用所见即所得的方式快速完成海报开发。

### 只需三步，即可完成海报开发

#### 一、启动服务

```bash
docker run -it --name fast-poster -p 5000:5000 tangweixin/fast-poster
```
